Notebook App
A simple console-based notebook built with Python (OOP)
Feathures:
  1. Add/Edit/Delete pages
  2. Auto page number
  3. Date & time for each page
  4. Clean menu system
  5. Utils for helper function
  6. Validator for checking inputs
Files:
  1. main.py: runs the program
  2. menu.py: handles menu
  3. notebook.py: manages pages
  4. page.py: page modle
  5. utils.py: helper tools
  6. validator.py: input checks
